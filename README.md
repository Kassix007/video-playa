# m3u8-player-proxy

A simple Node.js proxy + Clappr HTML player for streaming `.m3u8` with `Referer` spoofing to bypass 403 errors.

## How to Use

1. Run `npm install`
2. Run `npm start`
3. Open `http://localhost:3000`
